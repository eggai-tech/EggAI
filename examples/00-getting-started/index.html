Setup Instructions ‚è≥

Step 1: Create a Virtual Environment (Optional but Recommended) üåç

To avoid dependency conflicts, create and activate a virtual environment:

python -m venv venv
source venv/bin/activate  # For Windows: venv\Scripts\activate

Step 2: Install the EggAI Library üéì

Use pip to install the EggAI framework:

pip install eggai

Step 3: Start Required Services with Docker üö¢

EggAI relies on brokers like Kafka or Redpanda and other services to function seamlessly. Start these services using Docker Compose:

docker compose up -d

This command will: - Spin up the Redpanda broker. - Start the necessary infrastructure components for the framework.

Running the Example üèÜ

The example code for handling two agents is included in the examples/ folder. Run the main.py file to execute the demo:

python main.py

What happens: - Agent 1 sends a message to Agent 2. - Agent 2 processes the message and sends a response. - The framework handles message passing, retries, and logging.

Congratulations! You've successfully run your first EggAI Multi-Agent application. üéâ

Stopping and Cleaning Up ‚ùå

When you're done, stop and clean up the Docker containers to free up resources:

docker compose down -v

This will stop all running containers and remove any associated volumes.
